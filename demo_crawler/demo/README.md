# Web Spider Demo

## 说明

这个 demo 主要为了练习网页抓取

功能模块

- processer 负责管理爬虫的基本逻辑
- fetcher 负责解决代理IP，超时等问题
- strategy 负责处理每次爬取失败后的策略
